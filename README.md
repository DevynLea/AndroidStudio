# Rock Paper Scissors - AndroidStudio App

A simple and interactive Rock-Paper-Scissors games built using Kotlin and Android Stuido. The user plays against the computer, and the first to reach 10 points wins the match!

# Features

- Play Rock, Paper, Scissors against the computer
- Computer moves are randomly generated
- Score tracking for both player and CPU
- Game ends automatically when someone reaches 10 points
- Clean UI and responsive layout

# Tech Stack

- Kotlin
- Android Studio
- XML layouts
- Android SDK
